# 시뮬레이터 사진첩에 이미지 저장하기

시뮬레이터 사진첩의 파일 경로

```markdown
user(사용자) - (사용자이름) - 라이브러리 - Developer - CoreSimulator - Devices - 기기identifier - data - Media - DCIM - 100APPLE
```

* 라이브러리 폴더는 숨겨져 있으므로 `command + shift + .`으로 숨김파일을 볼 수 있도록 한다.
