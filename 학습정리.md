3. Nil Coalescing Operator("??")

물음표 하나는 옵셔널을 의미하며, 물음표 두개는 3중 조건 연산자를 의미한다.

```swift
a >= 0 ? "양수":"음수"
```

위의 코드는 a가 0보다 크거나 같다는 참의 조건일 경우 "양수"값을 택하고 반대로 0보다 작다면 거짓인 "음수"값을 택한다.

```swift
let value = a ?? b
=
let value = a != nil ? a! : b
```

6. describing

리눅스 컴파일러는 String과 NSString간의 형변환을 할 수 없다. 그래서 NSString의 init(string:)을 사용하여 변환한다.

```swift
let str = "hello world"

// String -> NSString
let strCast:NSString = NSString(string: str)

// NSString -> String
let result:String = String(describing: strCast)
```

7. 구조체와 클래스

구조체의 인스턴스는 값, 클래스의 인스턴스는 참조.

- 구조체 struct

구조체 명은 대문자, 프로퍼티와 메서드는 소문자

```swift
struct BasicInformation {
  var name: String // 저장 프로퍼티
  var age: Int 	   // 저장 프로퍼티
}

var yagomInfo: BasicInfromation = BasicInformation(name: "Yagom", age: 99)
yagomInfo.age = 100
yagomInfo.name = "Seba"

sebaInfo.age = 100 // 변경 불가 오류!
```

​	연산 프로퍼티 : 값을 저장한 것이 아니라 특정 연산을 실행한 결과값을 의미,

​				    클래스, 구조체, 열거형에 쓰일 수 있다.

```swift
struct CoordinatePoint{
  // 저장 프로퍼티
  var x: Int
  var y: Int
  
  // 연산 프로퍼티
  var oppositePoint: CoordinatePoint {
    // 접근자
    get {
      return CoordinatePoint(x: -x, y: -y)
    }
    
    // 설정자
    set(opposite) {
      x = -opposite.x
      y = -opposite.y
    }
  }
}

var yagomPosition: CoordinatePoint = CoordinatePoint(x: 10, y: 20)

print(yagomPosition) // 10, 20
ptint(yagomPosition.oppositePoint) // -10, -20

yagomPosition.oppositePoint = CoordinatePoint(x: 15, y: 10)
print(yagomPosition) // -15, -10
```
