3. Nil Coalescing Operator("??")

물음표 하나는 옵셔널을 의미하며, 물음표 두개는 3중 조건 연산자를 의미한다.

```swift
a >= 0 ? "양수":"음수"
```

위의 코드는 a가 0보다 크거나 같다는 참의 조건일 경우 "양수"값을 택하고 반대로 0보다 작다면 거짓인 "음수"값을 택한다.

```swift
let value = a ?? b
=
let value = a != nil ? a! : b
```
